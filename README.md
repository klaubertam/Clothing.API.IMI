# Clothing.API.IMI 👗

Back-end i projektit **Dyqan Rrobash**, ndërtuar me ASP.NET Core Web API (C#).

## Si funksionon
- Hap projektin në Visual Studio
- Shtyp F5 për ta startuar
- API ekzekutohet në `http://localhost:5000`

## Endpoints
- `GET /api/clothingitems` — Merr të gjitha produktet
- `POST /api/clothingitems` — Shto produkt të ri
- `PUT /api/clothingitems/{id}` — Edito produktin
- `DELETE /api/clothingitems/{id}` — Fshi produktin
